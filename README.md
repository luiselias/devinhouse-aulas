### Fundamentos do HTML

* Objetivo de todo software **ajudar pessoas**. E a interface é parte fundamental do software
* HTML, CSS e JS são ferramentas e como tais devem ser usadas em um contexto adequado e a web é seu *habitat natural*
* Criada na década de 90 por Tim Berners-Lee, icialmente para a ajudar a comunicação e a troca de pesquisas entre universidades
* HTML (HiperText Markup Language), uma liguagem que marcação de hipertexto que facilita a leitura e compreensão por máquinas e pessoas.
* HiperTexto, um documento que possibilita a navegação em outros documentos
* As Tags são o elemeto básico dos documentos HTML. Um documento é formado pela a união de várias Tags que estruturam e dão sentido ao documento
